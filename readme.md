# MIT 课程《计算机科学教育中缺失的一课》简体中文翻译

------------

> 源字幕文件来源于 Youtube 发布的公开网络内容。

Missing Semester 是 MIT 的一系列公开课，内容聚焦于讲授计算机专业实用的软件工具、开发技巧等。

这一系列课程涵盖从命令行、文本编辑器、版本控制到代码调试等内容。

目前，我们尝试做一些微小的努力，把这一系列课程的中文翻译带给大家。
